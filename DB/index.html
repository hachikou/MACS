MACS.DB ドキュメント

SQLデータベースにアクセスするためのユーティリティライブラリです。

サンプルプログラム

  • DBを使ったデータベースアクセスプログラム作成例

データベース接続

  • DBCon : DB接続を取り扱うクラス。 ADO.NETのDbConnectをもっと簡便に扱えるようにラッピングしたもの。
  • DBConPool : DBConのプールを提供するクラス
  • DBReader : DBテーブルアクセス: DbDataReaderのラッパークラス

データベース定義

  • DBTableDef : データベーステーブル定義
  • DBColumnDef : DBテーブルのカラム定義
  • DBIndexDef : DBテーブルのインデックス定義

SQL発行

  • DBTable : DBテーブルアクセスユーティリティクラス
  • DBCondition : SQL検索条件管理クラス
  • DBTableJson : DBTableとNDJsonを結びつける拡張メソッド

その他
