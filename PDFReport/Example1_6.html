枠線

枠線を引く方法をご説明します。

Example1_6.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <font name="default" size="14" />
  <font name="title" size="24" />

  <text x="50%" y="0mm" align="center" font="title">枠線のサンプル</text>

  <text x="20mm" y="20mm" width="50mm">枠線指定無し</text>
  <text y="+10mm" width="50mm" border="0.2mm">枠線あり</text>
  <text y="+10mm" width="50mm" border-top="0.2mm">上線あり</text>
  <text y="+10mm" width="50mm" border-bottom="0.2mm">下線あり</text>
  <text y="+10mm" width="50mm" border-left="0.2mm" border-right="0.2mm">左右線あり</text>
  <text y="+10mm" width="50mm" border="0.2mm" border-bottom="0.5mm">下線だけ太い</text>

  <box y="+10mm" width="50mm" height="10mm" border="0.2mm" />
  <box y="+15mm" width="50mm" height="10mm" border="0.2mm" crossline="0.2mm"/>
  <box y="+15mm" width="50mm" height="10mm" border="0.2mm" slashline="0.2mm"/>
  <box y="+15mm" width="50mm" height="10mm" border="0.2mm" backslashline="0.2mm"/>

  <box x="50%" y="150mm" width="20mm" height="20mm" border="0.1mm"/>
  <box x="50%" y="150mm" width="20mm" height="20mm" align="right" border="0.3mm"/>
  <box x="50%" y="150mm" width="20mm" height="20mm" vertical-align="bottom" border="0.6mm"/>
</document>

結果は、Example1_6.pdfをご覧下さい。

text タグに border 属性をつけると、文字列の周りに線を描画することができます。 border 属性の値は、枠線の太さです。

border-top , border-bottom , border-left , border-right 属性を使うと、それぞれ上線、下線、左線、右線だけを描画することができます。 また、上下左右のどれかに異なる太さを指定する事もできます。 もちろん、4つの線にそれぞれ異なる太さを指定することもできます。

なお、枠線を描画する場合には、必ず width 属性で幅を指定して下さい。

box タグは、任意の枠線を描画するものです。 box タグも、 x y 属性で左上隅の座標を指定し、 width 属性で幅を指定することができます。 さらに、 height 属性で高さを指定します。 border 属性の指定の仕方は、 text タグと全く同じです。

box タグまたは text タグに crossline 属性を指定すると、対角線を結ぶバッテンの線を描画することができます。

また、左下から右上の斜め線だけを引きたい場合には slashline 属性を使います。 また、左上から右下の斜め線だけを引きたい場合には backslashline 属性を使います。

box タグには align 属性を指定することができます。 align="right"とすると、X座標はボックスの左側ではなく、右側の位置を指定する意味になります。 align="center"とすると、X座標はボックスの中央を指定する意味になります。

同様に、 box タグに vertical-align 属性を指定することができます。 vertical-align="bottom"とすると、Y座標はボックスの上側ではなく、下側の位置を指定する意味になります。 vertical-align="middle"とすると、Y座標はボックスの中央を指定する意味になります。
