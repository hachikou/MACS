CSVファイルを表形式表示する

CSVファイルからデータを読み込み、表形式の表示をすることができます。

この場合、今までの例で使ったような、表部分以外のデータを定義するファイルと、表部分用のCSVファイルの2つのデータファイルをPDFReportに与えます。

埋め込みデータファイル1: example.ini 

会社名=PDF商事
住所=東京都北区赤羽
電話=03-0123-4567
FAX=03-0123-4568

埋め込みデータファイル2: example.csv 

#年,月,予算,実績
2011,1,853000,801100
2011,2,720000,748000
2011,3,811000,820900
2011,4,830000,815800
2011,5,835000,921000
2011,6,821000,908300
2011,7,804000,807600
2011,8,729000,732000
2011,9,813000,758800
2011,10,838000,799900
2011,11,843000,867100
2011,12,801000,884700

帳票フォーマットファイル: Example1_12.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <font name="default" size="12" />
  <font name="title" size="24" />

  <text x="50%" y="0mm" align="center" font="title">年間業績表</text>
  <text x="100%" y="0mm" align="right">{currentdate}</text>

  <box x="50%" y ="15mm" width="100mm" height="20mm" align="center" border="0.3mm">
    <text x="2mm" y="2mm">会社名: {会社名}</text>
    <text y="+1.5line">TEL:{電話} FAX:{FAX}</text>
  </box>

  <table x="50%" y="40mm" align="center" width="100mm" rows="12" border="0.3mm">
    <body height="1.5line" border-bottom="0.1mm">
      <text x="1mm" y="1mm" width="15mm">{list.年:postfix(/)}{list.月:fixlength(2)}</text>
      <text x="+30mm" width="30mm">{list.予算:decimal:postfix(円)}</text>
      <text x="+40mm" width="30mm">{list.実績:decimal:postfix(円)}</text>
    </body>
  </table>

</document>

実行方法: 

% PDFReport Example1_12.xml example.ini example.csv -o Example1_12.pdf

結果は、Example1_12.pdfをご覧下さい。

CSVファイルから取り込んだデータは、{list.項目名}という指定で参照します。

CSVの項目名は、ファイルの1行目にかかれたコメント行 ('#'で始まる行) で定義します。 もし、CSVファイルの1行目に項目名定義が無い場合は、{list.項目番号}と、名前ではなく番号で項目を指定して下さい。

表の描画には、 table タグを用います。 table タグの属性で、表の位置と幅、行数などを指定します。 （表の高さを height 属性で指定することもできますが、行数の制御がうまくゆかなくなるケースがありますので、できるだけ高さは指定せずに行数を指定して下さい。）

table タグ内には、 body タグを1つ作り、その中に1行分の表示指示のタグを記述します。 body タグには height 属性で1行の高さを指定して下さい。 また、 body の border-bottom ( border-top でも可)属性で、行を区切る横線の太さを指定できます。

body タグ内では、各行の左上隅が基準となる座標で文字描画などを指定します。

この例では body タグ内に{list.項目名}項目だけが使われていますが、一般の{項目名}を使うこともできます。 この場合、各行に同じ値が描画されることになります。

表の描画で最も重要な点は、与えたCSVファイルの行数（レコード数）が、表の行数指定よりも多い場合には、自動的に複数ページで描画されるという事にあります。 この際、表以外の部分は全てのページで同じものが描画されます。（ページ番号の例外を除く）。

例えば、Example1_10.xml 内のテーブルで、「rows="12"」としている部分を「rows="10"」に変更して実行すると、Example1_12_2.pdfのような、2ページ立てのPDFファイルになります。
