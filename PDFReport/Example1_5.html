文字の種類

font タグを使って文字の種類を指定することもできます。

Example1_5.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <basefont file="ipag.ttc,0" />
  <font name="default" size="18" />
  <font name="title" size="24" />
  <font name="mincho" size="18" file="ipam.ttc,0" />
  <font name="courier" size="18" file="Courier" />
  <font name="courier-b" size="18" file="Courier-Bold" />
  <font name="courier-o" size="18" file="Courier-Oblique" />
  <font name="courier-bo" size="18" file="Courier-BoldOblique" />
  <font name="helvetica" size="18" file="Helvetica" />
  <font name="helvetica-b" size="18" file="Helvetica-Bold" />
  <font name="helvetica-o" size="18" file="Helvetica-Oblique" />
  <font name="helvetica-bo" size="18" file="Helvetica-BoldOblique" />
  <font name="times" size="18" file="Times-Roman" />
  <font name="times-b" size="18" file="Times-Bold" />
  <font name="times-i" size="18" file="Times-Italic" />
  <font name="times-bi" size="18" file="Times-BoldItalic" />

  <text x="50%" y="0mm" align="center" font="title">FONT SAMPLE</text>
  <text x="20mm" y="20mm">デフォルト (IPAゴシック)</text>
  <text y="+10mm" font="mincho">明朝 (IPA明朝)</text>
  <text y="+10mm" font="courier">Courier</text>
  <text y="+10mm" font="courier-b">Courier Bold</text>
  <text y="+10mm" font="courier-o">Courier Oblique</text>
  <text y="+10mm" font="courier-bo">Courier BoldOblique</text>
  <text y="+10mm" font="helvetica">Helvetica</text>
  <text y="+10mm" font="helvetica-b">Helvetica Bold</text>
  <text y="+10mm" font="helvetica-o">Helvetica Oblique</text>
  <text y="+10mm" font="helvetica-bo">Helvetica BoldOblique</text>
  <text y="+10mm" font="times">Times Roman</text>
  <text y="+10mm" font="times-b">Times Bold</text>
  <text y="+10mm" font="times-o">Times Italic</text>
  <text y="+10mm" font="times-bo">Times BoldItalic</text>
</document>

結果は、Example1_5.pdfをご覧下さい。

font タグに file 属性をつけると、文字の種類（字体）を指定することができます。 file 属性に指定するのは、TrueTypeフォントファイル名です。 フォントファイルは、PDFReportコマンドが入っているディレクトリ内の "fonts" というディレクトリ内に入れておいて下さい。 標準では、IPAで提供しているフリーのフォントである、ipag.ttc (IPAゴシック)と ipam.ttc (IPA明朝)の2つだけが入っています。

一般のTrueTypeフォントファイルを用いる際には、 file 属性にはファイル名そのものを指定して下さい。 ipag.ttc ファイルのような、TrueTypeフォントコレクションファイルを用いる際には、ファイル名の後に ",0" を付けて下さい。 （TTCファイル内のセカンダリフォントを使う場合には ",1" を付けて下さい。）

basefont タグを用いると、 font タグで file 属性を持たないものに対するデフォルトのフォントファイルを指定することができます。 basefont タグは必ず font タグよりも前に宣言して下さい。

PDFで標準とされている、次の14種類のフォントについては、 font 属性のファイル名としてフォント名を指定すれば（フォントファイルが無くても）使うことができます。 

Courier, Courier-Bold, Courier-Oblique, Courier-BoldOblique,
Helvetica, Helvetica-Bold, Helvetica-Oblique, Helvetica-BoldOblique,
Times-Roman, Times-Bold, Times-Italic, Times-BoldItalic,
Symbol, ZapfDingbats

ただし、これらのフォントは日本語には対応していません。 また、SymbolとZapfDingbatsは記号用のフォントで、通常の英数字の表示もできません。
