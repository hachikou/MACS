テキストをボックスでまとめる

Example1_6では box タグを枠線を引く目的で使いましたが、 box タグには描画要素をまとめるという用途もあります。

Example1_7.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <font name="default" size="12" />
  <font name="title" size="24" />
  <font name="small" size="9" />
  <font name="large" size="18" />

  <text x="50%" y="0mm" align="center" font="title">テキストをボックスでまとめる</text>

  <text x="10mm" y="10mm">テキスト1</text>
  <text x="100%" y="20mm" align="right" font="small">テキスト2</text>

  <box x="20mm" y="30mm" width="100mm" height="150mm" border="0.1mm" font="large">
    <text x="10mm" y="10mm">テキスト3</text>
    <text x="100%" y="20mm" align="right" font="small">テキスト4</text>

    <box x="20mm" y="30mm" width="50mm" height="100mm" border="0.3mm">
      <text x="10mm" y="10mm">テキスト5</text>
      <text x="100%" y="20mm" align="right" font="small">テキスト6</text>
    </box>

  </box>

</document>

結果は、Example1_7.pdfをご覧下さい。

「テキスト3」と「テキスト4」が box タグの内側に入っている点に注意して下さい。 この場合、「テキスト3」の位置指定である「x="10mm" y="10mm"」は、 box の左上隅を起点とした座標になります。 つまり、「テキスト3」は実際には X座標 30mm、Y座標40mm の位置に描画されます。

さらに、 box を入れ子にする事もできます。

box に属性 font を付けると、その内部の text タグに font 属性を付けたのと同じ効果が得られます。

この例ではわかりやすいように box に枠線をつけましたが、もちろん枠線無しで box を使うこともできます。

若干トリッキーな例ですが、用紙の中心からの位置指定をしたい場合には、次のようにすると実現できます。 

  <box x="50%" y="50%" width="50%" height="50%">
    <text x="10mm" y="10mm">X</text>
  </box>

文字Xは、中心から右に10mm、下に10mmの位置に描画されます。

また、中心から左、上方向の位置を指定したい場合には、次のようにします。 

  <box x="0" y="0" width="50%" height="50%">
    <text x="!10mm" y="!10mm">Y</text>
  </box>

文字Yは、中心から左に10mm、上に10mmの位置に描画されます。
