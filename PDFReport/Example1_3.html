文字列の位置合わせと幅指定

text タグの align 属性と width 属性をご紹介します。

Example1_3.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <text x="50%" y="10mm" align="left">左あわせ</text>
  <text y="+10mm" align="center">中央あわせ</text>
  <text y="+10mm" align="right">右あわせ</text>

  <text y="+10mm">長い文字列を表示してみよう</text>
  <text y="+10mm" width="20mm">長い文字列を表示してみよう</text>
  <text y="+10mm" align="center" width="20mm">長い文字列を表示してみよう</text>
  <text y="+10mm" align="right" width="20mm">長い文字列を表示してみよう</text>
  <text y="+10mm" width="20mm" clip="yes">長い文字列を表示してみよう</text>
  <text y="+10mm" width="20mm" align="center" clip="yes">長い文字列を表示してみよう</text>
  <text y="+10mm" width="20mm" align="right" clip="yes">長い文字列を表示してみよう</text>
</document>

結果は、Example1_3.pdfをご覧下さい。

text タグに align 属性をつけると、指定したX座標が文字列の左端になる(align="left")、文字列の中央になる(align="center")、文字列の右端になる(align="right")ことを指定できます。 align 指定が無い場合は、"left"が指定されたものとみなされます。

タグに width 属性をつけると、文字列を描画する際の最大幅を指定することができます。 幅の指定には、X座標と同じ単位を用いることができます。 （"char"は文字幅を単位とするものですが、"X"の幅が基準になっているという点に注意して下さい。）

width 属性が指定されていると、その幅で収まらない文字は描画されません。 このとき、align="right" や align="center" が指定されていても、文字列の先頭部分が描画されるという点に注意して下さい。

width 属性を指定した際に、同時に clip="yes" 属性をつけると、文字単位でのクリッピングではなく、文字の途中でも切り取られます。 このときは、align="right"指定の場合文字列の最後の部分が、align="center"指定の場合文字列の中央が、それぞれ描画されます。
