文字の大きさ

標題などに大きさの違う文字を使いたい場合があります。 このようなときには、まず font タグを使って使用するフォントを宣言しておき、 text タグに font 属性を付けて利用します。

Example1_4.xml 

<?xml version="1.0" encoding="UTF-8"?>
<document>
  <font name="default" size="14" />
  <font name="title" size="30" />
  <font name="heightadjust" size="14" lineheight="18" />
  <font name="upper" size="14" lineheight="18" descent="-4" />
  <font name="lower" size="14" lineheight="18" descent="4" />

  <text x="50%" y="10mm" align="center" font="title">Font指定</text>

  <text x="10mm" y="+20mm">デフォルトフォント</text>
  <text y="+1line">デフォルトフォントで+1lineする。</text>
  <text y="+1line" font="heightadjust">行高さ指定フォントで+1lineする。</text>
  <text y="+1line" font="heightadjust">標準のベースライン</text>
  <text x="+50mm" font="upper">ベースラインを上に</text>
  <text x="+50mm" font="lower">ベースラインを下に</text>
</document>

結果は、Example1_4.pdfをご覧下さい。

font タグで、この文書で使用するフォントに名前を付けて宣言します。 name 属性で任意の名前を指定します。 size 属性で文字の大きさを指定します。単位はポイントです。

text タグで文字を描画する際に、 font 属性をつける事によって、 font タグで宣言したフォントを用いることができます。 font 属性が指定されていない text は、"default"という名前のフォントが指定されたものとみなされます。

font タグの属性として、 lineheight を指定することができます。 これは、Y座標の指定をする際の "line"単位の大きさを指定することになります。 line単位を全く用いない場合には、 lineheight 属性はあまり意味を持ちません。

font タグの属性として、 descent を指定することができます。 descent に正の数値を指定すると、文字列は下めに描画されます。 逆に負の数値を指定すると、文字列は上めに描画されます。

なお、 font タグは必ず text タグで使用するよりも前に宣言して下さい。 document の最初の部分で全て宣言する事をお勧めします。
